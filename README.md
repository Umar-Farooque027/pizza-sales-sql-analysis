# 🍕 Pizza Sales Analysis using MySQL

## 📌 Project Overview

This project focuses on analyzing pizza sales data using SQL to generate business insights and improve decision-making.

## 🎯 Objectives

* Analyze total number of orders
* Calculate total revenue generated
* Identify top-selling pizzas
* Understand customer preferences
* Track revenue trends over time

## 🛠️ Tools Used

* MySQL
* SQL Queries
* Canva (for presentation)

## 📊 Key Insights

* Retrieved total orders placed
* Calculated overall revenue
* Found top 3 pizzas based on revenue
* Analyzed category-wise sales
* Calculated percentage contribution of each pizza type
* Generated cumulative revenue trends

## 📂 Files Included

* SQL Queries File
* Dataset (CSV)
* Project Presentation (PDF)

## 🚀 Conclusion

This project helped in understanding how SQL can be used to analyze real-world sales data and generate actionable insights.
